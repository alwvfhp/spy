spy
===
<i>谁是卧底起源于湖南卫视综艺节目《快乐大本营》</i><br/>
<b>规则</b><br/>
1.刚开始每一个参赛选手有一个词,其中有一个的和其他的不一样被认为是卧底<br/>
2.每个人轮流发出一段描述(要求和自己词有关的),然后每个人投票(你认为谁嫌疑最大的)<br/>
3.票多着被认定出局(如果是卧底游戏结束，否则继续)<br/>
4.游戏剩余两人卧底还没找出来，卧底获胜，否则其他获胜<br/>
